# Linux Permissions Lab

## Goal
Practice Linux file permissions and basic commands.

## Commands used
- ls -l
- chmod 640 project_m.txt
- chmod g-rw project_m.txt
- grep, find

## Tasks completed
- Removed group read/write permissions
- Checked permissions with ls -l
- Managed hidden files

## Skills demonstrated
- Understanding rwx permissions
- Using octal notation
- File security best practices
